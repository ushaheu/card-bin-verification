# card-bin-verification
**Author:** [Ushahemba Ukange](https://github.com/ushaheu)  

**Framework:** Spring Boot

**Java Version:** 8  

**Build Tool:** Maven  

**Build Command**
```sh
mvn clean package
```
**Run Command**
```sh
mvn spring-boot:run
```
**Port:** 8080  

**Dependency**  
The following services must be running  
```sh
Zookeeper 
Kafka  
Redis  
```
